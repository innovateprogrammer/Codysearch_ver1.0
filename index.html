Frequently Asked Questions

Here are some of the most commonly asked questions.

  • What should we do to change the text colour?

    In order to change the text color just type color and then the name of the color your want eg: color orange

  • Why do we need to change search engine ?

    Every search engines have thier own performace status.Some search engine dominantly perform better than other providing more accurate and relevant results.

  • Why choose bing search for this, why not other like Firefox, Microsoft edge etc?

    Mozilla and Edge are Web Browsers and not search Engines.Both things are very different.Bing is one of those search engines that provides more versatile search results than any other.

  • If we choose the bing is of affect for that particular time or it's permanently?

    It will be set as the default browser for that particular session until you either restart CodySearch or change the search engine manually to chrome.

  • It is possible that we can use the google assistant to enter the command in our command prompt?
